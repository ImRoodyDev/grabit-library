<h1>Grabit Libray</h1>
Provider library that uses [grabit-engine](../grabit-engine) to scrape websites and extract multilingual streaming sources (movies, series, and live TV).
</div>

## Features

- **Movies & Series** — providers for English, Spanish, French, and multi-language sources
- **Live TV** — live stream providers
- **Subtitles** — dedicated subtitle providers
- **Extractors** — built-in support for Filemoon, DoodStream, MixDrop, Supervideo, and more

## Project Structure

| Folder                  | Purpose                                      |
| ----------------------- | -------------------------------------------- |
| `providers/media/`      | Movie & series providers grouped by language |
| `providers/live/`       | Live TV providers                            |
| `providers/subtitle/`   | Subtitle providers                           |
| `providers/extractors/` | Stream host extractors                       |
